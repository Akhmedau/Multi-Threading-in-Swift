GCD –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –∑–∞–¥–∞—á –Ω–∞ —É—Ä–æ–≤–Ω–µ —Å–∏—Å—Ç–µ–º—ã, —á—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–µ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–Ω–æ–≥–æ–ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–Ω—ã–µ –∏ –º–Ω–æ–≥–æ—è–¥–µ—Ä–Ω—ã–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–∞.

##### –û—Å–Ω–æ–≤–Ω—ã–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ –∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã GCD

GCD –æ—Å–Ω–æ–≤–∞–Ω –Ω–∞ –∫–æ–Ω—Ü–µ–ø—Ü–∏—è—Ö –æ—á–µ—Ä–µ–¥–µ–π (dispatch queues) –∏ –∑–∞–¥–∞—á (work items), –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–∑–≤–æ–ª—è—é—Ç —É–ø—Ä–∞–≤–ª—è—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ–º –∫–æ–¥–∞ –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ –∏–ª–∏ —Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ, —Å–µ—Ä–∏–π–Ω–æ –∏–ª–∏ –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ.

##### –û—Å–Ω–æ–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã GCD:

1.Dispatch Queues (–û—á–µ—Ä–µ–¥–∏ –∑–∞–¥–∞—á):

- Serial Queues (–°–µ—Ä–∏–π–Ω—ã–µ –æ—á–µ—Ä–µ–¥–∏): –í—ã–ø–æ–ª–Ω—è—é—Ç –∑–∞–¥–∞—á–∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –æ–¥–Ω–∞ –∑–∞ –¥—Ä—É–≥–æ–π.
- Concurrent Queues (–ü–∞—Ä–∞–ª–ª–µ–ª—å–Ω—ã–µ –æ—á–µ—Ä–µ–¥–∏): –í—ã–ø–æ–ª–Ω—è—é—Ç –∑–∞–¥–∞—á–∏ –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –≤ –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ–º —Ä–µ–∂–∏–º–µ.

  2.Dispatch Groups (–ì—Ä—É–ø–ø—ã –∑–∞–¥–∞—á): –û–±—ä–µ–¥–∏–Ω—è—é—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–∞–¥–∞—á –∏ –ø–æ–∑–≤–æ–ª—è—é—Ç –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –∏—Ö –∑–∞–≤–µ—Ä—à–µ–Ω–∏–µ.

  3.Dispatch Semaphore (–°–µ–º–∞—Ñ–æ—Ä—ã): –£–ø—Ä–∞–≤–ª—è—é—Ç –¥–æ—Å—Ç—É–ø–æ–º –∫ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ–º—É —Ä–µ—Å—É—Ä—Å—É.

  4.Dispatch Sources (–ò—Å—Ç–æ—á–Ω–∏–∫–∏ —Å–æ–±—ã—Ç–∏–π): –ü–æ–∑–≤–æ–ª—è—é—Ç —É–ø—Ä–∞–≤–ª—è—Ç—å –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã–º–∏ —Å–æ–±—ã—Ç–∏—è–º–∏, —Ç–∞–∫–∏–º–∏ –∫–∞–∫ —Ç–∞–π–º–µ—Ä—ã, —Ñ–∞–π–ª–æ–≤—ã–µ –¥–µ—Å–∫—Ä–∏–ø—Ç–æ—Ä—ã –∏–ª–∏ —Å–∏–≥–Ω–∞–ª—ã.

  5.Dispatch Work Items (–†–∞–±–æ—á–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã): –û–±—ä–µ–∫—Ç—ã, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–µ –∑–∞–¥–∞—á–∏, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –±—ã—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω—ã –≤ –æ—á–µ—Ä–µ–¥–∏.

##### –¢–∏–ø—ã –æ—á–µ—Ä–µ–¥–µ–π –≤ GCD

–û—á–µ—Ä–µ–¥–∏ –∑–∞–¥–∞—á ‚Äî —ç—Ç–æ –æ—Å–Ω–æ–≤–Ω–∞—è –∫–æ–Ω—Ü–µ–ø—Ü–∏—è GCD. –û—á–µ—Ä–µ–¥–∏ —É–ø—Ä–∞–≤–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ–º –∑–∞–¥–∞—á –∏ –æ–ø—Ä–µ–¥–µ–ª—è—é—Ç, –∫–∞–∫ —ç—Ç–∏ –∑–∞–¥–∞—á–∏ –±—É–¥—É—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è (–∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ –∏–ª–∏ —Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ, —Å–µ—Ä–∏–π–Ω–æ –∏–ª–∏ –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ).

-**DispatchQueue.main**: –≠—Ç–æ —Å–µ—Ä–∏–π–Ω–∞—è –æ—á–µ—Ä–µ–¥—å, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–ø–æ–ª–Ω—è–µ—Ç –∑–∞–¥–∞—á–∏ –≤ –≥–ª–∞–≤–Ω–æ–º –ø–æ—Ç–æ–∫–µ. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –∑–∞–¥–∞—á, —Å–≤—è–∑–∞–Ω–Ω—ã—Ö —Å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–º –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–æ–º (UI), —Ç–∞–∫–∏—Ö –∫–∞–∫ –æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ —ç–∫—Ä–∞–Ω–æ–≤ –∏–ª–∏ –æ–±—Ä–∞–±–æ—Ç–∫–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–≥–æ –≤–≤–æ–¥–∞.

-–£—Ä–æ–≤–Ω–∏ QoS:

**.default**: –°—Ä–µ–¥–Ω–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é.
**.utility**: –ù–∏–∑–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –¥–ª—è —Ñ–æ–Ω–æ–≤—ã—Ö –∑–∞–¥–∞—á, —Ç–∞–∫–∏—Ö –∫–∞–∫ –∑–∞–≥—Ä—É–∑–∫–∞ –¥–∞–Ω–Ω—ã—Ö –∏–ª–∏ –æ–±—Ä–∞–±–æ—Ç–∫–∞.
**.background**: –°–∞–º—ã–π –Ω–∏–∑–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –¥–ª—è –∑–∞–¥–∞—á, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è –≤ —Ñ–æ–Ω–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä, –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—è.
